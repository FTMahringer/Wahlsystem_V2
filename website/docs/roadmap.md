# Roadmap

## Recently completed

- aligned shared election types
- shared wizard system
- election creation/edit wizard
- first new election modes:
  - binary choice
  - approval voting
  - limited vote
  - Borda count
- adaptive voter ballots and result views
- admin user-management foundation
- student/teacher onboarding wizard

## Next focus

### User targeting and provisioning

- class/group assignment support
- token generation or access provisioning from admin flows

### User-management refinement

- editing student and teacher profiles
- optional delete/archive behavior
- better duplicate/conflict handling for bulk imports

### Advanced election roadmap

- candidate pair voting
- dual-role elections
- two-round runoff
- instant runoff voting
- score voting
- richer templates and result visualizations
